I have a problem with the main menu, how can I solve it?
If you have any problems with the main menu, you can fix it with the /menufix command.
How do I set up a Squad?
The Squad system is only available in Free Lobby mode. After logging into the mode, you can create a squad with the /squad command. After creating your squad, you can send invitations to your friends with /invite id.
How can I customize a character?
First, log in to the Settings page. Then scroll down to the bottom and click on the Costume Settings button and customize your character as you wish.
How do claims work?
The game mods in the Default Damaged Mods category are built with GTA's default damage, colloquially known as head damage. Game mods in the Custom Damaged Mods category are created with fully customized damage, colloquially known as body damage.
How do I get all the indicators back?
If you are not satisfied with the settings adjustments you have made, you can reset all your settings via the Reset button at the top by entering the Settings page.
How can I trick in deluxo mode?
After exiting the Safe Zone, you can trick via the B key.
